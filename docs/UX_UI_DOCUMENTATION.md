### Device Registration
- On successful login, device is automatically registered
- Device identifier stored in localStorage
- If device limit exceeded, oldest device logged out
- User sees notification about kicked device

### Security Features
- Password visibility toggle (eye icon)
- Remember me checkbox (7-day session)
- Failed attempt tracking
- Account lockout after 5 failures
